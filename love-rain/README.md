# 💕 cmatrix --love · Love Rain

黑客帝国风格，但下落的是 ❤️🌸💕，而不是绿色数字。

## 效果

全屏飘落爱心、花瓣、星光 — 温柔版 Matrix 代码雨。

## 五种模式

| 模式 | 内容 | 氛围 |
|------|------|------|
| 💕 全模式 | 爱心 + 花 + 星光 + 蝴蝶 + 丝带 | 浪漫大杂烩 |
| ❤️ 红心 | 各种爱心 emoji | 甜蜜暴击 |
| 🌸 花雨 | 樱花、玫瑰、雏菊 | 春日花见 |
| ✨ 星尘 | 星星、闪光、音符 | 梦幻夜空 |
| 🩷 粉彩 | 粉色系爱心+花 | 温柔纯爱 |

## 文件

| 文件 | 说明 |
|------|------|
| `love_rain.py` | Python turtle 版（纯内置库） |
| `index.html`  | Web Canvas 全屏交互版 |

## 使用

### Web 版

浏览器打开 `index.html`：
- 🖱️ **点击任意位置** → 绽放心动（12~24 个粒子炸开）
- ⌨️ **空格键** → 切换五种模式
- 🌫️ 拖尾效果 + 光晕
- 📱 移动端触摸支持

### Python 版

```bash
python3 love_rain.py
```

- 🖱️ **点击** → 绽放心动
- ⌨️ **空格** → 切换模式
- ⌨️ **Q** → 退出

## 粒子物理

- **正弦摇摆**：粒子左右飘荡，模拟花瓣/雪花飘落
- **变速下落**：每颗粒子独立速度
- **点击炸裂**：径向散开 + 微重力回落
- **旋转**：粒子缓慢自转

## 技术栈

- Python: 纯内置 `turtle`，零依赖
- Web: Canvas 2D + HSL 颜色空间 + 拖尾帧缓存
