# 💚 代码雨表白 · Code Rain Confession

Matrix 风格全屏代码雨 — 输入对方名字，代码雨落下，最终汇聚成那个名字。

## 效果流程

```
输入名字 → 全屏 Matrix 代码雨（~6秒）→ 雨滴汇聚 → 名字成形 + 浪漫揭示
```

### 三段式动画

| 阶段 | 效果 |
|------|------|
| 🌧️ **代码雨** | 数百字符从屏幕顶端倾泻，Matrix 经典绿 |
| 🧲 **汇聚** | 弹簧物理将每个雨滴拉向目标位置，颜色从绿渐变到浪漫粉 |
| ❤️ **成形** | 名字清晰呈现，光晕包裹，揭示文案浮现 |

## 技术

- **名字采样**：Canvas 渲染文字 → 像素采样 → 目标点阵
- **弹簧物理**：`F = -k·dx` + 阻尼，字符震荡归位
- **拖尾效果**：半透明黑色叠加帧缓存
- **颜色渐变**：RGB 线性插值 Matrix 绿 → 浪漫粉
- 全屏自适应，响应窗口大小变化

## 交互

- ⌨️ 输入名字 + Enter / 点击「开始」
- 🖱️ 点击画面提前触发汇聚（等不及 6 秒？）
- 🖱️ 成形后点击画面重新开始
- 🔄 底部「再来一次」按钮

## 使用

浏览器打开 `index.html`，零依赖。
