# 💚 代码雨表白 · Code Rain Confession

Matrix 风格全屏代码雨 — 输入对方名字，代码雨落下，最终汇聚成那个名字。

## 效果流程

```
输入名字 → 全屏 Matrix 代码雨（~6秒）→ 雨滴汇聚 → 名字成形 + 浪漫揭示
```

### 三段式动画

| 阶段 | 效果 |
|------|------|
| 🌧️ **代码雨** | 数百字符从屏幕顶端倾泻，Matrix 经典绿 |
| 🧲 **汇聚** | 弹簧物理将每个雨滴拉向目标位置，颜色从绿渐变到浪漫粉 |
| ❤️ **成形** | 名字清晰呈现，光晕包裹，揭示文案浮现 |

## 文件

| 文件 | 说明 |
|------|------|
| `code_rain.py` | Python turtle 版（纯内置库） |
| `index.html`  | Web Canvas 全屏交互版 |

## 使用

### Web 版

浏览器打开 `index.html`：
- ⌨️ 输入名字 + Enter / 点击「开始」
- 🖱️ 点击画面提前触发汇聚
- 🔄 成形后点击重来
- 📐 全屏自适应 + 拖尾效果 + 像素级名字采样

### Python 版

```bash
python3 code_rain.py              # 交互式输入名字
python3 code_rain.py 张三          # 直接指定名字
python3 code_rain.py 张三 8        # 名字 + 雨持续秒数
```

交互：
- `Enter` — 提前触发汇聚
- `R` — 重新开始
- `Q` — 退出

## 技术

### Web 版
- **名字采样**：Canvas 渲染文字 → `getImageData` 像素采样 → 目标点阵
- **拖尾效果**：半透明黑色叠加帧缓存
- **汇聚物理**：弹簧力 + 阻尼

### Python 版
- **目标网格**：按字符块划分目标区域，网格点阵拼字
- **光晕叠加**：多偏移 `turtle.write()` 实现发光效果
- **纯内置库**：turtle + random + math，零依赖

## 颜色渐变

`Matrix 绿 (0,255,65)` → 线性插值 → `浪漫粉 (255,120,150)`

汇聚过程中颜色平滑过渡，成形后定格在温暖粉色。
